# General

### Coding Style

- Use UNIX line returns
- Remove trailing white space
- Add new line at end of files

### Editor Settings

[Sublime Text] settings:

```json
{
    "ensure_newline_at_eof_on_save": true,
    "trim_trailing_white_space_on_save": true,
}
```

Also see [EditorConfig].
